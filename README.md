## 프로젝트 동기 및 목표

 - 코로나 사태로 인해 대면 영화 표 예매가 불가능한 상황에서 많은 고객이 자신이 원하는 영화표를 구매하기 어려워졌다. 
 - 이러한 상황을 해결해고자 지금의 프로젝트를 계획하게 되었고 이 프로젝트를 통해 개발된 온라인 영화표 예매 시스템을 통해서 많은 고객들이 자신이 원하는 영화표를 온라인 상에서 구할 수 있도록 하는 것이 이 프로젝트의 목표이다.


### 프로젝트 기능 구현 범위

1.	관람객 정보 관리 기능
  - 로그인
  - 로그아웃
  - 회원 가입 
2.  관람객의 영화 예매를 위한 기능
  - 영화 목록 조회 및 선택
  - 영화관 목록 조회 및 선택
  - 영화의 상영정보 조회 및 선택 선택
  - 좌석 선택
3.  관람객의 예매 정보 관리를 위한 기능
  - 현재 예매한 티켓 정보 조회
  - 현재 예매한 티켓 취소
4.  관리자를 위한 기능
  - 상영되는 영화의 정보 추가
  - 상영되는 영화의 정보 삭제
  - 상영되는 영화의 정보 수정
