# SWEC-Prominam

A full-stack online movie ticket reservation system developed for the **2021-2 Introduction to Software Engineering** course.

This project was designed to help customers reserve movie tickets online in a non-face-to-face environment, making it easier for users to search for and book the movie tickets they want.

## Project Background and Objective

Due to the COVID-19 situation, many customers faced difficulties purchasing movie tickets through traditional offline methods. As in-person booking became limited, the need for a convenient online reservation system became increasingly important.

To address this problem, we planned and developed this project with the goal of enabling customers to browse and book their desired movie tickets online in a simple and efficient way.

The main objective of this project is to provide an accessible **online movie ticket reservation system** that improves convenience for users and supports a smoother booking experience.

## Key Features

### 1. User Account Management
- User registration
- Login
- Logout

### 2. Movie Reservation Features
- Browse and select movies
- Browse and select theaters
- View and select screening schedules
- Select seats

### 3. Reservation Management
- View currently reserved ticket information
- Cancel current ticket reservations

### 4. Admin Features
- Add movie information
- Delete movie information
- Update movie information
